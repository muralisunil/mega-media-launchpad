# Mega Media — Static Landing Page

This repository now serves a single static webpage from [index.html](index.html).

## Run locally

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The generated static files are written to `dist/`.

## Edit content

- Main page markup and styles: [index.html](index.html)
- Logo asset: [public/logo.webp](public/logo.webp)
